**Snake** (Змейка)
-
- [X] Навправление движения (UP, DOWN, RIGHT, LEFT)

- [X] Элементы змейки (collections.deque из Элементов)

- [X] Найти элемент для новой головы
	- [X] Проверка что новая голова внутри змейки
	- [X] Проверка что новая голова внутри поля
- [X] Проверка что элемент внутри змейки
